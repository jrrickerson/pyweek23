Running the Game
----------------

Open a terminal / console and "cd" to the game directory and run:

  python3 run_game.py


How to Play the Game
--------------------
The instructions are given at the beginning of gameplay. Basically:
    arrow keys to move
    space to shoot
    < and > or , and . for side thrusters
    A or B when given a choice
    Return to advance the dialog
    ESC to quit
